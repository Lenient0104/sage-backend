# Sage-Backend

## Installation

### Prerequisites

1. Python >= 3.11
2. Poetry >= 1.1.11

### Installation

1. Clone the repository
2. `cd sage-backend`
3. `poetry install`

## Running the application

1. `cd sage-backend/backend`
2. `poetry run python manage.py runserver`

## Running the tests

1. `cd sage-backend/backend`
2. `poetry run python manage.py test`
